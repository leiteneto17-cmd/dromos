# Coleção de Skills — Framework v1.5.0 (atual)

Esta pasta é a versão consolidada e atual da biblioteca, em 03/07/2026. As 15 skills de papel originais estão completas, mais 2 de governança (17 no total). Comece por `README.md` se esta é uma sessão nova sem contexto da construção.

## Como instalar

**Claude.ai**: Customize > Skills > "+" > Create skill > Upload → suba cada .zip da pasta `skills/` individualmente.

**Claude Code**: extraia cada pasta (sem o .zip) para `.claude/skills/` no seu repositório. Coloque `governance/`, `cheatsheets/`, `README.md`, `FRAMEWORK_PHILOSOPHY.md` e `CHANGELOG.md` na raiz do repositório para servirem de referência do projeto inteiro, além da cópia do Library Standards já embutida em cada skill.
